Most Laravel tutorials teach concepts in isolation. You learn routing in one lesson, Eloquent in another, and authentication somewhere else, but you never see how they all connect inside a real application. This course takes a different approach.

You will build **Catatku**, a personal journal application, from an empty project to a fully working app across 12 structured lessons. Every concept is introduced at the moment you need it, applied immediately, and built upon in the next lesson. By the end, you will not just know Laravel syntax. You will understand why the code is organized the way it is.

The course follows Laravel 13 with PHP 8.3 and covers the modern conventions, including the `#[Fillable]` attribute for models. No starter kits or scaffolding. You write every line yourself, so nothing feels like magic.

**This course is for you if:**
- You understand basic PHP (variables, functions, arrays, conditionals) but have never used a framework
- You have tried Laravel before but got confused about how all the pieces fit together
- You want a portfolio-ready project, not just a collection of disconnected exercises

**By the end, you will have:**
- A working application you can demo, deploy, or extend on your own
- A solid mental model of how Laravel processes a request from URL to database and back
- The confidence to read the official Laravel documentation and understand it

No prior Laravel experience required. Lesson 2 walks you through the full environment setup on Windows.
